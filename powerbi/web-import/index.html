Manual CSV Import

  1. Open Power BI Desktop
  2. Click "Get Data" → "Text/CSV"
  3. Navigate to data/master/ folder
  4. Select and import these files in order:
  • dim_date.csv
  • dim_product.csv
  • dim_customer.csv
  • dim_employee.csv
  • fact_sales.csv
  • fact_budget.csv
  • fact_inventory.csv
  • dim_kpi_targets.csv

Power Query Script Import

Copy this script into Power BI's Advanced Editor:

let
    Source = Folder.Files("C:\YourPath\data\master"),
    FilteredFiles = Table.SelectRows(Source, each [Extension] = ".csv"),
    ImportedFiles = Table.AddColumn(FilteredFiles, "Data", 
        each Csv.Document([Content], [Delimiter=",", Encoding=1252, QuoteStyle=QuoteStyle.None])),
    RemovedColumns = Table.RemoveColumns(ImportedFiles, 
        {"Content", "Extension", "Date accessed", "Date modified", "Date created", "Attributes", "Folder Path"}),
    ExpandedData = Table.ExpandTableColumn(RemovedColumns, "Data", 
        Table.ColumnNames(RemovedColumns[Data]{0}))
in
    ExpandedData

Folder Import (Recommended)

  1. Click "Get Data" → "Folder"
  2. Browse to your data/master folder
  3. Click "Combine & Transform Data"
  4. Power BI will automatically detect all CSV files
  5. Review and click "OK"
Note: Update the folder path to match your local installation directory.

Sales → Employee

Fact_Sales[EmployeeKey] → Dim_Employee[EmployeeKey]

Budget → Date

Fact_Budget[DateKey] → Dim_Date[DateKey]

Inventory → Product

Fact_Inventory[ProductKey] → Dim_Product[ProductKey]

Key DAX Measures

Create these essential measures:

// Total Sales
Total Net Sales = SUM(Fact_Sales[NetSales])

// Year-over-Year Growth
Sales YoY Growth % = 
VAR CurrentSales = [Total Net Sales]
VAR LastYearSales = CALCULATE([Total Net Sales], SAMEPERIODLASTYEAR(Dim_Date[Date]))
RETURN DIVIDE(CurrentSales - LastYearSales, LastYearSales, 0)

// Profit Margin
Gross Profit Margin % = 
DIVIDE(SUM(Fact_Sales[GrossProfit]), SUM(Fact_Sales[NetSales]), 0)
Tip: Find all 60+ measures in powerbi/datasets/DAX_Measures.txt

Publish to Power BI Service

  1. Save your report: File → Save As → "Bevco_Executive_Dashboard.pbix"
  2. Click "Publish" in the Home ribbon
  3. Select your workspace (or create new)
  4. Wait for upload to complete
  5. Click "Open in Power BI" when done
